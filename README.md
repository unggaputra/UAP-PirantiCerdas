# UAP-PirantiCerdas
Alat penghangat kandang ayam, merupakan alat yang dibuat menggunakan ESP32 sebagai mikrokontroler, DHT22 sebagai sensor temperatur dan kelembaban, dan rellay untuk mengatur arus listrik terhadap penghangat. 

Alat ini juga dilengkapi dengan kemampuan kontrol melalui aplikasi Blynk yang terhubung ke internet. Fitur ini memungkinkan pengguna untuk mengatur pada suhu berapa alat penghangat harus menyala dan mati.
